## Estructura

- `inicio/`: punto de partida para resolver durante la clase.

## Ejecución sugerida

```bash
cp .env.example .env
export ANTHROPIC_API_KEY="tu_api_key"
python python/clase-10/final/main.py
```

manejo de errores y seguridad.
